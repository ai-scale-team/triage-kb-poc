// GitHub Settings

Cases are stored as cases/CASE-YYYY-NNN.json in the configured repo.

You need a Personal Access Token with repo scope (classic) or Contents: Read/Write (fine-grained).
→ github.com/settings/tokens

The token is stored only in your browser (localStorage) and sent directly to the GitHub API.
